# ⚡ 批量上传 - 快速指南

## 🎯 三步完成批量上传

### 1️⃣ 选择文件

```
点击"批量选择"卡片（紫色边框）
→ 在文件选择器中按住 Ctrl/Cmd 多选
→ 点击"打开"
```

### 2️⃣ 确认列表

```
查看文件列表
→ 取消不想上传的文件
→ 确认选择无误
```

### 3️⃣ 开始上传

```
点击"批量上传 (N)"按钮
→ 等待上传完成
→ 查看结果
```

---

## 📊 三种上传方式对比

| 方式 | 文件来源 | 何时使用 |
|------|---------|---------|
| 单文件 | 本地 | 1-2本书 |
| **批量选择** ⭐ | 本地 | 3-30本书 |
| 目录扫描 | 服务器 | 服务器上的书 |

---

## 💡 快速技巧

### 分类上传

```
1. 设置分类："小说"
2. 批量选择小说文件
3. 上传

4. 切换分类："技术"
5. 批量选择技术书籍
6. 再次上传
```

### 文件筛选

在文件选择器中：
- Windows: 文件类型 → "*.epub"
- Mac: 格式 → 选择类型

---

## 🎨 界面识别

**紫色边框** = 批量选择本地文件 ⭐  
**蓝色边框** = 单文件上传  
**绿色边框** = 服务器目录扫描

## ❓ 常见问题

**Q: 可以选择文件夹吗？**  
A: 不能，按住 Ctrl/Cmd 多选文件

**Q: 上传失败怎么办？**  
A: 失败的文件保留在列表，修复后重新上传

**Q: 速度慢吗？**  
A: 逐个上传，显示准确进度

---

## 🚀 最佳实践

**日常使用**（10-20本书）：
```
批量选择 → 确认列表 → 一键上传
```

**大量书籍**（100+本）：
```
分批上传，每次20-30本
按分类分批处理
```
